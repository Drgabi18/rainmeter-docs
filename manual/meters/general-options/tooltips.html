--- layout: manual title: 'Tooltips' ---

Creates a ToolTip which appears when the mouse is hovered over the meter.

Options

ToolTipTextRequired
The text to display.

Values from MeasureName from the meter can be used with %1, %2 etc. as appropriate for various meter types:
  • Line, String: %1, %2, %3, ...
  • Histogram: %1, %2
  • Others: %1

Note: To wrap the text on multiple lines, use the Built-In Variable #CRLF# to provide line breaks.
ToolTipTitle
Sets a bold title for the tooltip. Only one line of text can be used.
ToolTipIcon
Specifies an Icon to be placed in the tooltip. This can be the path to a .ico file or one of these preset icons:
  • INFO
  • WARNING
  • ERROR
  • QUESTION
  • SHIELD (Windows Vista and newer)

Note: ToolTipTile is required to use ToolTipIcon.
ToolTipTypeDefault: 0
Defines the style of the tooltip. Valid values are:
  • 0 Normal tooltip
  • 1 Ballon tooltip

Note: ToolTipType=1 is not supported on Windows XP.
ToolTipWidthDefault: 1000
Maximum width for the tooltip. When the width is reached, the text will automatically wrap.

Note: As a ToolTipTitle cannot be wrapped, do not set this width less than the length of the title, or there could be unexpected results.
ToolTipHiddenDefault: 0
If set to 1, the ToolTip is not displayed.

Note: ToolTipHidden can also be used in the [Rainmeter] section to hide all tooltips in the skin.
