--- layout: manual title: 'MediaKey plugin' ---

Plugin=MediaKey sends various keystrokes found on multimedia keyboards. It works with Spotify, Zune, foobar, and Windows Media Player. The plugin is controlled by the !CommandMeasure bang.

Arguments

  • NextTrack
  • PrevTrack
  • Stop
  • PlayPause
  • VolumeMute
  • VolumeDown
  • VolumeUp

Example

{% highlight ini %} [MeasureMediaKey] Measure=Plugin Plugin=MediaKey [MeterMute] Meter=String SolidColor=0,0,0,255 FontColor=255,255,255,255 Text=Mute LeftMouseUpAction=!CommandMeasure "MeasureMediaKey" "VolumeMute" {% endhighlight %}
