--- layout: snippets title: 'Convert Colors' ---

These functions can be used to convert Rainmeter color code formats from RGB to hexadecimal, and vice versa.

{% highlight lua %} function ConvertToHex(color) -- Converts RGB colors to HEX local hex = {} for rgb in string.gmatch(color, '%d+') do table.insert(hex, string.format('%02X',tonumber(rgb))) end return table.concat(hex) end -- ConvertToHex {% endhighlight %}
{% highlight lua %} function ConvertToRGB(color) -- Converts HEX colors to RGB local rgb = {} for hex in string.gmatch(color, '..') do table.insert(rgb, tonumber(hex, 16)) end return table.concat(rgb,',') end -- ConvertToRGB {% endhighlight %}
