Sorts a given table into a random order.

{% highlight lua %} function SortRandom(input) assert(type(input) == 'table', ('SortRandom must receive a table. Received %s instead.'):format(type(input))) math.randomseed(tonumber(tostring(os.time()):reverse():sub(1,6))) local temp = {} repeat table.insert(temp, table.remove(input, math.random(1, #input))) until #input == 0 return temp end -- SortRandom {% endhighlight %}

Note: Unlike table.sort this function does not directly manipulate the original table. The sorted table is returned as a result of the function.
