Measure=CPU measures CPU usage.

Options

General measure options
All general measure options are valid.
Processor Default: 0
If set to 0, measures the average of all CPU cores. If set to a number (1, 2, etc.), measures a specific CPU core.

Value

The value of a CPU measure is a percentage from 0 to 100.

Example

``` ini [Rainmeter] Update=1000 BackgroundMode=2 SolidColor=0,0,0,255 [MeasureAverageCPU] Measure=CPU [MeasureCPU1] Measure=CPU Processor=1 [MeterText] Meter=String MeasureName=MeasureAverageCPU MeasureName2=MeasureCPU1 X=5 Y=5 W=100 H=35 FontColor=255,255,255,255 NumOfDecimals=1 Text="Average: %1%#CRLF#Core 1: %2%" ```
