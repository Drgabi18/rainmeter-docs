Measure=String A measure which can be used to create or manipulate a string.

Options

General measure options

All general measure options except InvertMeasure are valid.

String

The string value of the measure.

Note: Any combination of hard-coded text, variables, dynamic variables, or section variables can be used. Formulas are not evaluated in this option.

Example

``` txt [Rainmeter] Update=1000 AccurateText=1 DynamicWindowSize=1 [MeasureMyString] Measure=String String=Red, Green, Blue RegExpSubstitute=1 Substitute="Green":"Yellow" [MeterMyString] Meter=String MeasureName=MeasureMyString FontColor=255,255,255,255 SolidColor=47,47,47,255 Padding=5,5,5,5 Text=The colors are %1 ```
