Measure=Plugin allows interactation with various DLL plugins specifically written to work with Rainmeter.

Options

General measure options
All general measure options are valid.
Plugin specific options
Plugins have their own specific options that must be placed in the measure section. For more information, refer to the Plugins entry for each.
Plugin
Name of the plugin.

Example

``` ini [Rainmeter] Update=1000 BackgroundMode=2 SolidColor=0,0,0,255 [MeasureUserName] Measure=Plugin Plugin=SysInfo SysInfoType=USER_NAME [MeterText] Meter=String MeasureName=MeasureUserName X=5 Y=5 W=100 H=25 FontColor=255,255,255,255 Text="User: %1" ```
