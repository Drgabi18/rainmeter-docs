This creates the clamp(n, lower, upper) mathematical function, which is not part of the standard Lua math libraries.

The argument n is the number or formula to test. lower is the lowest value it can be, and upper is the highest value it can be.

``` lua function math.clamp(val, lower, upper) assert(val and lower and upper, 'error: math.clamp(val, lower, upper)') return math.max(lower, math.min(upper, val)) end ```

Example:
This will constrain the value of myVal between 15 and 25.

``` lua for i = 10, 30 do myVal = math.clamp(i, 15, 25) end ```
