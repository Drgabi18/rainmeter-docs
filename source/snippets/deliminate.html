--- layout: snippets title: 'Deliminate' ---

Deliminates a given string into a table using the specified delimiter.

``` lua function Delim(line, delimiter) -- Separate String by single character delimiter assert(delimiter:len() == 1, 'Delim: Delimiter may only be a single character') local tbl = {} for word in line:gmatch('[^%' .. delimiter .. ']+') do table.insert(tbl, word) end return tbl end ```
