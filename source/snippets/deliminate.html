Method 1

This splits a given string into a table using a specified single character delimiter.

Example usage: splitTable=string.split('First part|Second part', '|')

``` lua function string.split(inString, inDelimiter) assert(inDelimiter:len() == 1, 'string.split: Delimiter may only be a single character') local outTable = {} for capturedString in inString:gmatch('[^%' .. inDelimiter .. ']+') do table.insert(outTable, capturedString) end return outTable end ```

Method 2

This splits a given string into a table using a specified single or multiple character delimiter.

Example usage: splitTable=string.split('First part||Second part', '||')

``` lua function string.split(inString, inPattern) local outTable = {} local findPattern = "(.-)" .. inPattern local lastEnd = 1 local currentStart, currentEnd, capturedString = inString:find(findPattern, 1) while currentStart do if currentStart ~= 1 or capturedString ~= "" then table.insert(outTable, capturedString) end lastEnd = currentEnd + 1 currentStart, currentEnd, capturedString = inString:find(findPattern, lastEnd) end if lastEnd <= #inString then capturedString = inString:sub(lastEnd) table.insert(outTable, capturedString) end return outTable end ```
