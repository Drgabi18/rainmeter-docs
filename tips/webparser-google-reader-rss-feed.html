--- layout: tips title: 'WebParser: Google Reader as RSS Feed' ---

Tip by poiru

With this method, one single WebParser measure (or a single RegExp, to be specific) can handle virtually all feeds. The idea is to use Google Reader to read feeds (be it Atom, RSS, FeedBurner, or something else) and output them with a single, constant format. We can then parse the various information using a single regular expression.

The address:
http://www.google.co.uk/reader/view/feed/#URL#?n=8

where #URL# can be any feed (e.g. http://newsrss.bbc.co.uk/rss/newsonline_world_edition/front_page/rss.xml). Note that if the URL contains special characters like ? or &, it must be encoded using a URL encoder that converts those characters into URL entities.

?n=8 tells Google Reader to retrieve 8 items.

The code below will return the following:

  • The title of the feed (e.g. BBC News)
  • The link to 8 items (if they exist -- the RegExp won't fail if less than 8 items are available)
  • The title to 8 items (if they exist)
  • The date of 8 items (if they exist)
{% highlight ini %} [Variables] URL=http://newsrss.bbc.co.uk/rss/newsonline_world_edition/front_page/rss.xml GET=.*(?(?=.*<div class="item">).*<div class="item">.*<a href="(.*)">(.*)</a>.*<div class="item-info">.* on (.*)</div>) [MeasureFeed] Measure=Plugin Plugin=WebParser Url=http://www.google.co.uk/reader/view/feed/#URL#?n=8 RegExp="(?siU)<h1>(.*)</h1>#GET##GET##GET##GET##GET##GET##GET##GET#" UpdateRate=1500 [MeasureItem1] Measure=Plugin Plugin=WebParser Url=[MeasureFeed] StringIndex=2 [MeasureLink1] Measure=Plugin Plugin=WebParser Url=[MeasureFeed] StringIndex=3 [MeasureItem2] Measure=Plugin Plugin=WebParser Url=[MeasureFeed] StringIndex=4 [MeasureLink2] Measure=Plugin Plugin=WebParser Url=[MeasureFeed] StringIndex=5 ... and so forth until [MeasureLink8] {% endhighlight %}

Note: Use google.co.uk to get the date in DD/MM/YY and google.com for MM/DD/YY)
